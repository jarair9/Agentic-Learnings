# Chapter 05: Tools And Function Calling

An AI model cannot directly check your files, search your database, or call your Python functions unless you give it tools.

A tool is a normal function in your app.

Function calling means:

1. You describe a tool to the model.
2. The model asks to call the tool.
3. Your Python code runs the tool.
4. You send the result back.
5. The model writes the final answer.

This is one of the most important agentic AI ideas.
