# Chapter 01: Your First AI Call

An AI model is like a very smart text prediction machine. You send it input, and it sends back output.

In this chapter, you learn the smallest useful program:

1. Create an OpenAI client.
2. Send a question.
3. Print the answer.

This is the seed of everything else. Even big agents start with one model call.

## Run

```powershell
python chapter_01_first_ai_call/main.py
```
