# Chapter 08: Mini RAG

RAG means Retrieval-Augmented Generation.

That sounds fancy, but the basic idea is simple:

1. Store useful notes.
2. Search the notes.
3. Give the best notes to the model.
4. Ask the model to answer using those notes.

This chapter uses a tiny keyword search so you understand the idea before using vector databases.

## Run

```powershell
python chapter_08_mini_rag/main.py
```
