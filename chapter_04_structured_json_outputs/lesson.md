# Chapter 04: Structured JSON Outputs

Normal AI text is nice for humans.

But programs prefer structured data, like JSON dictionaries and lists.

Agents often need structured output because your code must read the model's decision.

Example:

```json
{
  "topic": "agent loop",
  "difficulty": "beginner",
  "next_action": "teach"
}
```

## Run

```powershell
python chapter_04_structured_json_outputs/main.py
```
