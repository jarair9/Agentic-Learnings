# Chapter 15: Logging And Debugging

When an agent breaks, print statements can become messy.

Python's `logging` module gives you cleaner debugging.

Logs can show:

- when the agent starts
- which tool was called
- what arguments were used
- whether something failed

Good logs help you understand your agent without staring at the code for hours.

## Run

```powershell
python chapter_15_logging_and_debugging/main.py
```
